# Faculty Photos - Naming Convention

## Photo Format Instructions

To add photos for each faculty member, please follow this naming convention:

### Naming Rule
Name each photo file using the **email prefix** (the part before `@`) from their email address, followed by `.jpg` (or `.png`, `.jpeg`).

### Example
- Email: `godfreys@srmist.edu.in` → Filename: `godfreys.jpg`
- Email: `vijayak@srmist.edu.in` → Filename: `vijayak.jpg`

## File Location

- `.jpg` / `.jpeg` (recommended)

## Photo Specifications (Optional but Recommended)
- **Aspect Ratio:** Square (1:1) or portrait
- **Recommended Size:** At least 400x400 pixels
