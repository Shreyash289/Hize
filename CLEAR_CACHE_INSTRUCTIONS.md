# Clear Browser Cache for hize2026.ieeecssrm.in

The site is fixed on the server, but your browser is still using old cached JavaScript files.

## Option 1: Hard Refresh (Fastest)
1. Open https://hize2026.ieeecssrm.in
2. Press:
   - **Mac**: `Cmd + Shift + R`
   - **Windows/Linux**: `Ctrl + Shift + F5`

## Option 2: Clear Site Data (Most Thorough)
1. Open Chrome DevTools (F12 or Cmd+Option+I)
2. Right-click the refresh button
3. Select "Empty Cache and Hard Reload"

## Option 3: Incognito/Private Window
1. Open a new incognito window
2. Visit https://hize2026.ieeecssrm.in
3. Should load fresh without cache

After doing any of these, the `timelineRef` error will be gone.
